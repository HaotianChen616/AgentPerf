# AgentPerf

AI Agent 应用性能测试、负载分析、指标收集工具集。

## 目录

| 目录 | 说明 |
|------|------|
| [nanobot-workload/](nanobot-workload/) | 基于 nanobot-ai 的 CPU 超分负载测试 — 单核多实例并发，时延分解（LLM/工具/框架），CPU 利用率采样 |

## 快速开始

进入对应子目录查看各自的 README 和使用说明。

## 贡献

欢迎添加新的性能测试负载脚本，每个测试放在独立子目录下，包含：
- 测试脚本
- 结果分析/可视化脚本
- README（用法、依赖、结果解读）
- requirements.txt
